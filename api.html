Quick Start

Get started in minutes with our simple REST API. No complex setup required.

GET /api/v1/convert

Convert time between cities

Example Request:

curl "https://api.worldtimeconverter.com/v1/convert?from=new-york&to=london&time=2026-01-15T14:30:00" \ -H "Authorization: Bearer YOUR_API_KEY"

Example Response:

{ "success": true, "data": { "from": { "city": "New York", "timezone": "America/New_York", "time": "2026-01-15T14:30:00", "offset": "-05:00" }, "to": { "city": "London", "timezone": "Europe/London", "time": "2026-01-15T19:30:00", "offset": "+00:00" }, "difference": { "hours": 5, "minutes": 0, "description": "London is 5 hours ahead" } } }

API Endpoints

GET /api/v1/timezone/{city}

Get current time and timezone information for a city

GET /api/v1/cities

List all available cities in the database

POST /api/v1/duration

Calculate duration between two dates/times

GET /api/v1/dst/{city}

Get daylight saving time schedule for a city

Authentication

All API requests require authentication using a Bearer token. Include your API key in the Authorization header:

Authorization: Bearer your_api_key_here
